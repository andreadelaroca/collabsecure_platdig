# CollabSecure
## Contexto
La empresa ficticia TechNova Learning, dedicada a crear soluciones educativas digitales,
necesita una herramienta interna para que sus equipos puedan compartir tareas,
comunicarse y gestionar pequeños proyectos de manera segura.
Durante el último año, han sufrido problemas relacionados con:
- Uso de plataformas no autorizadas.
- Falta de prácticas de seguridad digital.
- Pobre coordinación entre los miembros del equipo.
El área de Innovación solicita a un grupo de estudiantes que diseñen una aplicación
funcional, sencilla pero útil, que simule un entorno de colaboración básico, aplicando
principios de trabajo en equipo digital, seguridad y ética.

## Objetivo del proyecto
Desarrollar una aplicación funcional (mínimo viable) que permita:
- Registrar tareas o mensajes entre usuarios.
- Mostrar información compartida en un tablero o lista.
- Incorporar al menos dos medidas de seguridad y ética (por ejemplo:
autenticación básica, manejo responsable de datos, control de acceso,
avisos de privacidad).
